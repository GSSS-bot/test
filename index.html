【1】安全書類の提出依頼

協力業者へ安全書類（グリーンファイル）の提出依頼を行う際の、現場・積算購買部・経理・建築事務の 役割分担と流れを省略せずすべて記載しています。上のカードだけでも流れは把握できますが、 下に詳細説明・例外・FAQを完全収録しています。

現場で対応

  • 積算購買へ見積提出（契約開始）
  • 経理の注文書送付＝契約完了を確認
  • 未提出の業者へ直接声かけ or 建築事務へ連絡
  • 安全書類が揃ったら入場可否を判断
建築事務がサポート
・契約完了を確認し、GSで企業招待＋提出依頼メール送付
・提出物の受領→初回チェック→不備修正依頼
・最新の進捗を管理表「安全書類確認状況」に反映・共有
注意
未契約（＝注文書未送付）の業者には依頼不可。誤依頼は、日程矛盾・書類やり直し・注文書再作成など大きな手戻りを招きます。

全体の流れ（ロング版）

  1. 契約開始（現場所長）：積算購買へ見積書を提出し、契約手続きを開始
  2. 業者追加（積算購買部）：管理表「安全書類確認状況」に業者が追加（契約日程に注意）
  3. 注文書送付（経理）：業者と現場所長へ注文書送付 → 契約完了
  4. 提出依頼（建築事務）：GSで企業招待＋提出依頼 → 受領・初回チェック・不備修正依頼

提出期限・状況確認

  • 提出期限の目安：入場の1か月前まで
  • 最新の依頼・提出状況：管理表「安全書類確認状況」で確認

提出を急ぎたい場合（催促）

  • 現場から直接連絡（電話／Teams）
  • 建築事務へ共有 → 事務側からも催促・調整を実施

催促の例文（コピペ可）

「◯◯工事の安全書類につきまして、提出が未確認です。
入場は◯/◯予定のため、本日中のご対応をお願いいたします。」

よくある質問（FAQ）

Q. 契約前でも書類依頼したい場合？

A. 以下すべてを満たす場合、例外的に可能：

  • 入場日と契約日程に矛盾がない（注文日 → 工期開始 → 入場日の順が保たれる）
  • 入場日の前倒しの可能性がない
  • 業者がGS加入済（別現場で提出履歴あり）
  • 現場から直接の声かけが可能

補足（背景）

GS運用は契約と連動します。契約未整備のまま依頼すると、入力済み書類の作り直しや注文書の再発行が発生します。 そのため未契約業者への依頼は原則不可としています。

早期契約にご協力ください

安全書類の提出依頼は契約完了（注文書送付）後のみ可能です。 早期契約は現場立ち上げの遅延や書類のやり直しを防ぎます。

現場で対応

  • 見積提出 → 契約手続きを早めに開始
  • 注文書送付（契約完了）を確認
  • 契約前入場が必要な場合は積算購買・建築事務へ事前相談
建築事務がサポート
・契約完了後のみGSで招待・提出依頼
・提出物のチェック／不備対応
・日程矛盾の有無を確認し、必要に応じて調整
未契約業者には依頼不可
未契約＝注文書未送付。
注文日 → 工期開始 → 入場日の順が崩れると、書類やり直し／注文書再作成／見積再発行の手戻りが発生します。

契約前依頼が可能なケース（例外）

  • 契約日程と入場日に矛盾なし
  • 入場の前倒し可能性なし
  • 業者がGS加入済（提出履歴あり）
  • 現場から直接依頼の声かけが可能

「台帳作成通知」の掲示

再下請負契約が発生する場合、建設業法第14条の3第1項に基づく元請への届出が必要です。 周知のため、GSによる書面通知＋現場掲示の両方が必須です。

現場で対応

  • 通知文を印刷し、指定位置に掲示
  • 仮囲い（外側）＋詰所／休憩所の2か所に掲示
建築事務がサポート
・雛形Excelの提供／内容整合の確認
・掲示場所の相談対応
注意
書面通知はGSで送付済み。掲示が無いと法令要件を満たしません。

手順（ロング版）

  1. 【安全書類雛形】現場ごと → 右クリックでExcelアプリで開く（ブラウザNG）
  2. シート「通知」を開き、オレンジ枠の内容を確認
  3. 掲示場所のサイズに合わせて2部印刷（標準A4）
  4. 仮囲い（外側）・詰所／休憩所に掲示
補足（掲示場所の推奨）
・仮囲い外側：通行人・検査員が確認しやすい
・詰所／休憩所：下請業者が確実に目にする

新規入場者アンケート提出

業者は入場前に作業員名簿の登録・提出が必要です。現場と事務の役割分担を明確化し、 運用上の例外・注意点まで含めて記載します。

現場で対応

  • 新規入場者教育の実施
  • アンケートの誤記・読みにくい字の修正（必要に応じ代筆）
  • スキャンして提出（入場日ごと／業者混在OK／毎日推奨）
建築事務がサポート
・名簿登録状況の確認／不備の是正依頼
・QR付アンケの運用支援、直行直帰などの特別対応の相談受付
注意
QR付アンケの「送り出し教育」欄は元請側で登録する項目のため、出力時は 「いいえ」表示で問題ありません。

手順（ロング版）

  1. 新規入場者教育を実施し、本人へ内容確認。相違があれば所属業者に従業員情報の更新を依頼
  2. アンケートをスキャンして提出（入場日ごとに整理／業者混在OK）

特別対応（例）

直行直帰等でQR付アンケの出力・持参が困難な場合：

  • 前日までにメール／FAXでアンケート送付などの柔軟対応可（現場判断でOK、事務も支援）

問い合わせ先（スキャン設定）

複合機の初期設定はJGS㈱が実施します。スキャン設定の不具合や変更がある場合はJGS㈱へ連絡し、 併せて建築事務へ状況共有をお願いします。

社会保険未加入者

作業員の社会保険加入は法令上の義務です。未加入者は原則入場不可で、状況によっては行政指導の対象となります。 ここでは検査での指摘を避けるための実務対応を整理します。

現場で対応

  • 入場予定者の加入状況を名簿で確認
  • 未加入の場合は所属業者へ是正依頼を促す
  • 掲示物・資料配布で加入推進を周知
  • 検査時は「是正中／指導中」である旨を説明
建築事務がサポート
・加入状況更新の依頼送付／名簿情報の確認・是正管理
・加入促進の資料提供（掲示・配布）
注意
法令上の義務のため、未加入作業員は入場不可です。

検査対応（背景）

名簿で加入状況を確認されるため、未加入が残っていると指摘対象になります。 次の取り組みで、是正完了までの間の指摘回避につながった事例があります。

指摘回避の実例

  • 「本社にて対応・指導中」である旨を説明する
  • 国交省ガイドライン等の加入促進資料の掲示・配布を提示する

Edge記憶のパスワード削除

Edge に保存された不要なログイン情報（ID/パスワード）を削除し、誤ログインを防ぐ手順です。

① この業務の目的

不要になった Edge のパスワードを削除し、誤アカウントでのログインを防ぐための操作です。

② 手順

  1. 設定を開く：右上メニュー →「設定」
  2. パスワード管理：「パスワードとオートフィル」→「Microsoft パスワードマネージャー」
  3. 対象のIDを選択：例）グリーンサイト＝ kensetsu-site.com
  4. Windowsセキュリティ認証：PCログインパスワードを入力
  5. 削除
【補足】よくある質問
・保存パスワードが多いほど誤ログインが起きやすくなります。
・共有利用の現場PCでは、定期的な整理がおすすめです。
